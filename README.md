# 🏛️ NagarSeva — AI-Powered Citizen Services Platform

> Smart Governance & Civic Tech | Hackathon Project

A unified AI-powered platform enabling citizens to access government services, file grievances, and track resolutions — in multiple languages.

## 🚀 Live Features

### 1. 🤖 AI Grievance Intelligence System
- Submit complaints in plain language
- AI (Claude) auto-classifies category, priority (High/Medium/Low)
- Routes to the correct government department
- Generates unique ticket ID

### 2. 📊 Predictive Governance & Analytics Dashboard
- Real-time complaint metrics
- Weekly trends bar chart
- Department-wise donut chart
- AI-powered predictions for upcoming issues

### 3. 💬 Multilingual AI Chatbot
- Powered by Claude AI
- Supports: English, Hindi, Rajasthani, Gujarati, Tamil
- Quick-action buttons for common queries
- Full conversation history

### 4. 📍 Real-Time Complaint Tracking
- Enter ticket ID to see full resolution timeline
- Step-by-step status with timestamps

## 🛠️ Tech Stack

- **Frontend:** Pure HTML, CSS, JavaScript (no framework needed)
- **AI:** Anthropic Claude API (claude-sonnet-4-6)
- **Design:** Dark theme, responsive, mobile-friendly

## ⚡ Quick Start

1. Clone the repo:
```bash
git clone https://github.com/PRAKHAR-7806/nagarseva-citizen-platform.git
```

2. Open `index.html` in your browser — that's it!

> **Note:** For AI features (grievance analysis & chatbot), the Anthropic API is called directly. In production, proxy API calls through a backend to keep your key secure.
